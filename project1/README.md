# Project 1：机械臂建模与正运动学验证

## 一、项目简介

本项目为《机器人》课程设计的第一个阶段任务，主要围绕机械臂的运动学建模、正运动学求解及工具箱验证展开。项目目标是帮助同学建立从机械臂结构理解、参数建模、齐次变换推导，到代码实现与结果验证的完整认识。

本项目既是一个独立任务，也是后续路径生成、轨迹插值、仿真测试等项目的基础代码框架。因此，在完成本项目时，应尽量保持代码结构清晰、函数接口统一、注释规范，为后续继续扩展做好准备。

---

## 二、任务要求

每个小组需在以下两种机械臂型号中任选一种开展建模与实现：

- 络石 CR7
- 络石 SR3

围绕所选机械臂，完成以下内容：

1. 根据题目提供的机械臂结构图、坐标系定义及参数信息，建立正运动学模型。
2. 在课程设计报告中给出完整的坐标系设置、参数表、齐次变换矩阵推导过程。
3. 使用 Matlab 编写代码，实现所选机械臂的正运动学计算。
4. 使用 RTB 工具箱建立对应机械臂模型，并利用 `fkine()` 对自编写结果进行验证。
5. 理解和测试齐次变换矩阵、旋转矩阵、RPY 角、欧拉角之间的关系。

## 三、代码框架说明

本项目提供统一的 Matlab 模板代码，建议目录结构如下：

project1_robot_model/
├── main_project1_test.m
├── robot_params.m
├── build_mdh_table.m
├── mdh_transform.m
├── forward_kinematics.m
├── build_rtb_robot.m
├── pose_to_rpy_eul.m
├── test_fk_basic.m
├── test_fk_vs_rtb.m
└── test_pose_conversion.m

各文件功能说明如下：

### 1. `main_project1_test.m`
项目总入口文件。运行本文件可依次执行基础正运动学测试、RTB 对比测试以及位姿表示转换测试。

### 2. `robot_params.m`
机械臂参数文件。用于根据所选型号返回机械臂结构参数，是整个项目的统一参数入口。

### 3. `build_mdh_table.m`
根据关节变量和机械臂参数生成参数表。后续正运动学计算会调用本文件。

### 4. `mdh_transform.m`
根据单节参数计算齐次变换矩阵，是运动学建模中的核心基础函数。

### 5. `forward_kinematics.m`
用于串联所有关节变换，得到末端执行器相对于基坐标系的位姿。

### 6. `build_rtb_robot.m`
基于 RTB 工具箱建立机械臂模型，用于调用 `fkine()` 进行对比验证。

### 7. `pose_to_rpy_eul.m`
用于将齐次变换矩阵转换为 RPY 角和欧拉角，帮助理解不同位姿表示方法。

### 8. `test_fk_basic.m`
进行零位测试和简单关节角测试，用于检查正运动学函数是否基本可用。

### 9. `test_fk_vs_rtb.m`
将自编写的正运动学结果与 RTB 的 `fkine()` 结果进行对比。

### 10. `test_pose_conversion.m`
测试位姿表示之间的转换，帮助理解姿态表达相关知识。

## 四、同学需要完成的内容

本项目中的部分代码已经搭好框架，但仍保留了若干 `TODO`，需要同学根据所选机械臂的实际参数和自己的建模结果完成填写。主要包括以下部分：

### 1. 补全机械臂参数
在 `robot_params.m` 中，根据 CR7 或 SR3 的实际结构参数补全以下内容：

- `a`
- `alpha`
- `d`
- `offset`

注意，这些参数必须与课程设计报告中的参数表保持一致。

### 2. 完成参数表组织
在 `build_mdh_table.m` 中，根据所采用的 DH 或 MDH 建模方法，正确组织参数表。

### 3. 补全单节齐次变换矩阵
在 `mdh_transform.m` 中，根据报告中的公式推导完成齐次变换矩阵表达。

### 4. 检查正运动学结果
在 `forward_kinematics.m` 中理解和检查累计变换过程，确认末端位姿求解逻辑正确。

### 5. 建立 RTB 机器人模型
在 `build_rtb_robot.m` 中根据所采用的参数定义建立 RTB 机械臂模型，并用于结果验证。
